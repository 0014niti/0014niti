# Nnotes 

Nnotes is a modernized, beautiful handwriting and sketching app for Android, built with Kotlin and Jetpack Compose.

## Features
- **Infinite Canvas**: Sketch, write, and draw without limits.
- **Claymorphism UI**: A beautiful, soft 3D user interface that is easy on the eyes.
- **Floating Tool Palette**: A customizable, drag-and-drop tool palette that stays out of your way.
- **Interactive 3D Mascots**: Choose a profession-based companion (Student, Doctor, etc.) that hangs on your toolbar and keeps you motivated!

## Building the App
Open the project in Android Studio, or run the following command from the root directory:
`ash
./gradlew assembleDebug
`
Then, install it on your device:
`ash
adb install -r app/build/outputs/apk/debug/app-debug.apk
`
