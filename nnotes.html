Key Features

🔍
Off-Main-Thread Deep Zoom

A background C/C++ engine redraws high-resolution viewports off the UI thread. Zoom in endlessly and ink strokes stay razor sharp.

🖊️
Stylus & Touch Aware

Pen and finger inputs are processed separately. Pan and pinch-zoom with your finger while drawing smoothly with your stylus.

🖍️
Real Multiply Highlighting

Highlighters composite live every frame with true multiply blending, so overlapping strokes deepen naturally like real ink.

📄
Smart PDF Dark Mode

Annotate PDFs with ease and invert page colors for comfortable night reading while leaving embedded photos untouched.

🔒
Local Storage (SAF) Privacy

No accounts, no cloud tracking, and zero telemetry. Uses Android's Storage Access Framework for complete file privacy.

⚡
Open Vector (.nnote) Format

Strokes remain editable vector objects. Re-select, move, recolor, or erase any mark at any time without flattening to bitmaps.

Technical Architecture

C / C++ Native Engine (NDK)
Custom C++ stroke trees, CMake build integration, and off-thread viewport rendering for low-latency ink response.
Kotlin & Jetpack Compose
Modern declarative UI for Android 8.0+ targeting SDK 36, delivering smooth touch gestures and dark mode UI.
PDFBox Android
On-device PDF rendering, vector layer compositing, and night-mode page color inversion.
